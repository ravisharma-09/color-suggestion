# ColorSuggestor

Simple project to generate random colors.

I made this because while building websites it feels difficult to choose good colors. This tool helps quickly get color ideas and use them easily.
<img width="2704" height="1518" alt="image" src="https://github.com/user-attachments/assets/67468f31-7f5a-499a-816a-376ce8014adb" />

Features
Shows multiple color cards
Each card has hex value
Copy button copies the color
Apply button sets background
Resuggest button creates new colors

Tech
HTML CSS JavaScript

Run
Open index.html in browser

Made for practice and to solve a real small problem while designing websites
